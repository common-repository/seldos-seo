== Description ==

Manage all of your SEO settings of your Wordpress site with a single plugin created by SEO experts.

With this plugin you can,

- Maintain Wp-json security
- Perform SSL activation and 301 redirects
- Add title, description and canonical tag to your content and pages
- Create automatic alt tags for your images
- Implement Google Analytics and Search Console Activation/Setup
- Implement Yandex Metrica Setup

Plugin will be updated regularly.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `seldos-seo.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
